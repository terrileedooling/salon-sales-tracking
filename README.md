# Sales Tracker

A full-stack sales tracking web application built with React, Firebase Authentication, and Firestore. 
I designed it for small businesses to track products, sales, suppliers, and gain real-time insight into their data.

ðŸ”— **Live Demo:** [https://sales-tracker-3dc35.web.app](https://sales-tracker-3dc35.web.app)


## Features

- User authentication  and user creation with Firebase Auth (Email/Password)
- Add, edit, delete, and view products, sales, suppliers, and categories
- Sales cart with stock management
- Discounts, taxes, and total calculations
- Data syncing with Firestore
- Production-ready Firebase hosting setup
- Dashboard feature which allows user insights into what is happening

## Tech Stack

- Frontend: React, Vite, JavaScript, CSS
- Backend: Firebase Firestore & Authentication
- Hosting: Firebase Hosting
